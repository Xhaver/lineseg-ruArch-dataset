# Чеклист подготовки чертежа

Рабочий порядок: исходный DWG → `full_annotated.dxf` → `geometry_only.dxf`

---

## 1. full_annotated.dxf

### Геометрия
- [ ] Блоки взорваны (`EXPLODE` → повторить до исчезновения блоков)
- [ ] Вся геометрия присутствует: стены, двери, окна, колонны
- [ ] Сантехника / мебель: оставить как есть ИЛИ удалить (на твоё усмотрение)

### Аннотации (автоопределяются скриптом)
- [ ] TEXT / MTEXT на месте
- [ ] DIMENSION / MULTILEADER / LEADER на месте
- [ ] ATTRIB из блоков доступны (блоки уже взорваны)

### Ручная разметка (LWPOLYLINE вокруг области → нужный слой)
- [ ] Штамп / основная надпись → слой `YOLO_STAMP`
- [ ] Рамка листа → слой `YOLO_FRAME`:
  - **2 полилинии** (внешняя + внутренняя) — скрипт сам вычислит 4 полосы
  - **1 полилиния** — bbox используется напрямую
- [ ] Таблицы из линий (экспликации, ведомости) → слой `YOLO_TABLE`
- [ ] Номера помещений (кружок + цифра): **разметить** на `YOLO_TABLE` ИЛИ **удалить**

### Убрать
- [ ] HATCH (штриховки)
- [ ] Вьюпорты, рамки пространства листа

### Сохранение
- [ ] Формат: DXF R2010+
- [ ] INSUNITS = 6 (метры)
- [ ] Имя файла: `N_full_annotated.dxf` → в папку `sources/`

---

## 2. geometry_only.dxf  *(делается из full_annotated)*

### Удалить
- [ ] TEXT, MTEXT, DIMENSION, LEADER, MULTILEADER
- [ ] HATCH
- [ ] Полилинии разметки: `YOLO_TABLE`, `YOLO_STAMP`, `YOLO_FRAME`
- [ ] Штамп и рамку листа (линии)
- [ ] Номера помещений (если не удалены ранее)
- [ ] Обозначения: направление подъёма лестниц, оси, отметки уровней
- [ ] Сантехника / мебель (если решил убрать)

### Привести к единому виду
- [ ] `EXPLODE` всего → повторить до исчезновения блоков
  - **Исключение:** CIRCLE не взрывать (AutoCAD не умеет; скрипт сделает сам)
- [ ] Всё перенести на слой `0`
- [ ] Цвет: `ByLayer` или явно чёрный/белый

### Сохранение
- [ ] Формат: DXF R2010+
- [ ] INSUNITS = 6 (метры)
- [ ] Имя файла: `N_geometry_only.dxf` → в папку `sources/`

---

## 3. Проверка после сохранения

```bash
# Проверка размеров растра (без создания файлов)
python scripts/make_crops.py sources/N_geometry_only.dxf --scale 100 --units m --dry-run
```

- [ ] Растр в разумных пределах: **1500–6000 px** по каждой стороне
- [ ] Кропов ожидается: **3–8** (если меньше 3 — мало геометрии)

```bash
# Препроцессинг и проверка блоков
python scripts/preprocess_dxf.py sources/N_geometry_only.dxf --units m
```

- [ ] `INSERT(exploded): 0` — блоки полностью взорваны
- [ ] `INSERT: 0` — нет необработанных блоков
- [ ] Итого LINE разумное количество: **500–5000**

---

## Быстрая справка по параметрам

| Параметр | Значение |
|---|---|
| INSUNITS | 6 (метры) |
| --units | `m` |
| --scale | `100` (для масштаба 1:100) |
| --dpi | `300` (по умолчанию) |
| 1 ед. DXF | 1 м = 1000 мм |
| px/м при 1:100, 300 DPI | **118 пикс/м** |
| Стена 4 м → | **~473 px** |
| Стена 0.2 м (перегородка) → | **~24 px** |
